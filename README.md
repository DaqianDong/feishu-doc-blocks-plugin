# 飞书云文档检查器插件

一个飞书云文档插件，用于统计文档内容，了解文档结构。

## 功能特性

### 文档统计
- **字数统计**：统计文档总字数，区分友好块和问题块
- **Token 估算**：预估文档的 Token 数量
- **块类型分析**：统计各类块的数量

### 智能分类
- **友好块**：文本、标题、列表、引用等常规内容块
- **问题块**：白板、任务、OKR、议程等特殊内容块

### 表格管理
- **表格统计**：统计文档中的表格数量
- **快速定位**：支持在表格间循环跳转

### 内容导航
- **问题块跳转**：快速定位到问题块位置

### 界面设计
- **折叠设计**：默认折叠状态，节省空间
- **实时更新**：文档编辑时自动更新统计数据

## 安装

### 环境要求
- Node.js >= 14
- npm >= 6

### 安装依赖
```bash
npm install
```

## 使用

### 开发模式
```bash
npm start
```

启动后，在飞书文档中打开插件调试链接即可使用。

### 生产构建
```bash
npm run build
```

### 上传发布
```bash
npm run upload
```

## 功能说明

### 块类型分类

插件将飞书文档的块类型分为两类：

**友好块**（适合 AI 处理）：
- 文本、标题（1-5级）
- 列表（有序、无序）
- 引用、待办
- 代码块、分割线
- 图片、表格、文件
- 其他常规内容块

**问题块**（可能影响 AI 处理）：
- 标题（6-9级）
- 内嵌网页、云文档小组件
- 任务、OKR
- 白板、议程
- AI 模板
- 降级块（无权限或已删除）

### 版本管理

插件版本号通过 `package.json` 统一管理，遵循语义化版本规范（SemVer）：
- 主版本号（Major）：不兼容的 API 修改
- 次版本号（Minor）：向下兼容的功能性新增
- 修订号（Patch）：向下兼容的问题修正

## 配置说明

### 修改版本号

编辑 `package.json` 文件中的 `version` 字段：

```json
{
  "version": "1.0.0"
}
```

### Webpack 配置

插件使用 Webpack DefinePlugin 自动注入版本号到代码中。

## 开发指南

### 添加新的块类型统计

1. 在 `src/App.tsx` 中定义新的块类型
2. 更新 `UNFRIENDLY_BLOCK_TYPES` 或友好块分类
3. 在 `getBlockTypeName` 函数中添加中文翻译
4. 测试验证统计结果

### 样式定制

修改 `src/index.css` 文件，主要样式类：
- `.wordcount-demo` - 主容器
- `.stats-column` - 统计栏
- `.block-list` - 块列表
- `.scroll-btn` - 跳转按钮
